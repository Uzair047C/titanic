# Titanic Survival Prediction

This repository contains a beginner-friendly machine learning project for the Kaggle Titanic dataset.

## Files

- `train.csv` - training dataset
- `test.csv` - test dataset
- `gender_submission.csv` - sample submission file
- `predict.ipynb` - notebook for data analysis and prediction

## Project Description

The goal is to predict passenger survival on the Titanic using the provided tabular features.
The notebook includes exploratory analysis, preprocessing, model training, and prediction workflow.

## Usage

1. Open `predict.ipynb` in Jupyter/VS Code.
2. Run cells in order.
3. Review predictions and modify model steps as needed.

## Dataset Source

- Kaggle Titanic: Machine Learning from Disaster
